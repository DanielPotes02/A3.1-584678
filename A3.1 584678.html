Dado que se realizan mltiples pruebas (una por cada gen), se aplicaron tres mtodos distintos para controlar el error de tipo I asociado a pruebas mltiples, utilizando un umbral de significancia de = 0.05:

from scipy.stats import ttest_ind
from statsmodels.stats.multitest import multipletests

# Prueba t para cada gen entre clases 2 y 4
t_stats = []
p_values = []

for gene in X.columns:
    stat, p = ttest_ind(X_2[gene], X_4[gene], equal_var=False)
    t_stats.append(stat)
    p_values.append(p)

# Correccin de mltiples pruebas
alpha = 0.05

# Bonferroni
_, bonf_pvals, _, _ = multipletests(p_values, alpha=alpha, method='bonferroni')
bonf_significant = np.array(X.columns)[bonf_pvals < alpha]

# Holm
_, holm_pvals, _, _ = multipletests(p_values, alpha=alpha, method='holm')
holm_significant = np.array(X.columns)[holm_pvals < alpha]

# Benjamini-Hochberg (FDR)
_, bh_pvals, _, _ = multipletests(p_values, alpha=alpha, method='fdr_bh')
bh_significant = np.array(X.columns)[bh_pvals < alpha]

print("Genes significativos con Bonferroni:", bonf_significant)
print("Genes significativos con Holm:", holm_significant)
print("Genes significativos con Benjamini-Hochberg:", bh_significant)

Con el propsito de extender el anlisis comparativo a las cuatro clases de cncer presentes en el conjunto de datos, se realiz un experimento similar al anterior, pero utilizando el anlisis de varianza (ANOVA). A diferencia de la prueba t, que compara medias entre dos grupos, ANOVA permite evaluar si existen diferencias significativas en las medias de ms de dos grupos simultneamente.

Para llevar a cabo esta prueba, se utiliz la funcin f_oneway del mdulo scipy.stats, la cual requiere como entrada los valores de expresin gnica estratificados por clase. Por lo tanto, se organiz la informacin del conjunto de datos separando las muestras de cada clase (1, 2, 3 y 4) y agrupando las expresiones gnicas correspondientes a cada gen.

El resultado de la prueba ANOVA para cada gen consiste en un estadstico F y un p-value asociado, el cual indica la probabilidad de observar una variabilidad tan grande entre grupos simplemente por azar. De nuevo, dado que se realizan mltiples pruebas (una por gen), se aplicaron correcciones por comparaciones mltiples para mantener el control del error de tipo I.

In[5]:
from scipy.stats import f_oneway

# Obtener datos estratificados por clase
grouped_data = [X[y == cls] for cls in sorted(y.unique())]

# Prueba ANOVA para cada gen
anova_pvalues = []
for gene in X.columns:
    gene_values_by_class = [group[gene] for group in grouped_data]
    _, p = f_oneway(*gene_values_by_class)
    anova_pvalues.append(p)

# Correccin Benjamini-Hochberg como ejemplo
_, anova_corrected, _, _ = multipletests(anova_pvalues, alpha=0.05, method='fdr_bh')
significant_genes_anova = np.array(X.columns)[anova_corrected < 0.05]

print("Genes con diferencias significativas entre las 4 clases (ANOVA):")
print(significant_genes_anova)
# imprimir numeros de genes significativos
print("Nmero de genes significativos:", len(significant_genes_anova))

# Funcin para evaluar y mostrar mtricas
def evaluar_modelo(nombre_modelo, modelo, X_test, y_test):
    y_pred = modelo.predict(X_test)
    
    print(f"\n=== Resultados para {nombre_modelo} ===")
    print(f"Accuracy: {accuracy_score(y_test, y_pred):.4f}")
    print("\nReporte de clasificacin:")
    print(classification_report(y_test, y_pred, digits=4))
    
    # Matriz de confusin

    plt.title(f'Matriz de Confusin - {nombre_modelo}')
    plt.xlabel('Etiqueta Predicha')
    plt.ylabel('Etiqueta Verdadera')
    plt.show()

La seleccin de las variables utilizadas como entrada al modelo se realiz antes de separar los datos en entrenamiento y prueba. Esto significa que los modelos podran estar aprovechando informacin que normalmente no deberan tener al momento de entrenarse.

En contextos con pocos datos, es posible que los modelos se ajusten demasiado a las caractersticas especficas de esas muestras, lo que tambin puede inflar artificialmente el rendimiento.
